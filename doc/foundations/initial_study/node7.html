Domain subscription

A user of the Project Architect may decide to subscribe to a domain of the Project, that is to be notified of any incoming information item corresponding to this domain, with respect to security issues (Some users may not be allowed to see everything).

More generally, users shall be able to register many aspects of the project life cycle (requests for features, progress, releases, problem management).
