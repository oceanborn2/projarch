Document shell

A document shell is a data structure allowing to handle modifications to a document over time. Basically, a document can be revised (minor modifications) or completely rewritten (major modifications). A document shell is designed to help handle these situations by providing a shell (some sort of document container, sheltering a unique document, including all the revisions, whichever way they have been implemented).
