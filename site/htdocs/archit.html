Implementation issues

The Project Architect uses Java and J2EE as a development platform. We chose this platform both for speed of development and for the huge API that comes with it. On the other hand, it has been decided to use code generation in order to enforce consistent coding guidelines (on top of a framework) and to speed up development. I am currently investigating this issue using perl and The Template Toolkit
